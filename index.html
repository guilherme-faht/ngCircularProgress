ngCircularProgress Directive

Default Properties

Uses the default colors.

<percent-display percent="21" size="60" ></percent-display>

Change the colors

Uses the property "colors" to change the colors.

<percent-display percent="4" size="60" colors="#e5ebee,#66879c,#ffffff" ></percent-display>

Change the size

Uses the property "size" to change the size.

<percent-display percent="19" size="70" colors="#8a4047,#691520,#ffffff" ></percent-display>

Change the percent

Click the bottom to change the percent value.

<percent-display percent="83" size="70" colors="#8a4047,#691520,#ffffff" ></percent-display>
